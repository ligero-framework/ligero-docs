Ligero

A lightweight Java web framework for modern applications.
Zero dependencies. Virtual threads. Expressive API.

Get Started GitHub
App.java 
import com.ligero.Ligero;

public class App {
    public static void main(String[] args) {
        Ligero app = Ligero.create(8080);

        app.get("/hello/{name}", ctx ->
            ctx.json(Map.of("message",
                "Hello, " + ctx.pathParam("name") + "!")));

        app.start();
    }
}

Features

Blazing Fast

Zero dependencies in the core. Virtual threads. Trie-based router resolves ~140ns with 300+ routes.

Expressive API

Express-inspired handler API. Clean routes, middleware pipeline, and error handling.

Secure by Default

OWASP-aligned security headers, request hygiene, and CSRF enabled out of the box.

Pluggable SPIs

Server engines, JSON mappers, templates — swap via ServiceLoader without code changes.

Compile-time DI

Optional annotation processor generates wiring at compile time. No reflection, no scanning.

Observable

Micrometer metrics, OpenTelemetry tracing, structured logs, and a devtools dashboard.
