Ligero

A lightweight Java web framework for modern applications.
Zero dependencies. Virtual threads. Expressive API.

Get Started GitHub
App.java 
import com.ligero.Ligero;

public class App {
    public static void main(String[] args) {
        Ligero app = Ligero.create(8080);

        app.get("/hello/{name}", ctx ->
            ctx.json(Map.of("message",
                "Hello, " + ctx.pathParam("name") + "!")));

        app.start();
    }
}

Features

Blazing Fast

Zero dependencies in the core. Served on Java 21 virtual threads. Trie-based router resolves in ~140ns with 300+ routes.

Expressive API

Express-inspired handler API. Define routes, middleware, and error handlers with clean, readable code.

Secure by Default

OWASP-aligned security headers, request hygiene, and CSRF protection enabled automatically. Safe out of the box.

Pluggable SPIs

Server engines, JSON mappers, template engines, tracing — all via ServiceLoader. Swap without code changes.

Compile-time DI

Optional annotation processor generates explicit wiring at compile time. No reflection, no classpath scanning.

Observable by Design

Per-route Micrometer metrics, OpenTelemetry tracing, structured access logs, and a live devtools dashboard.
