Ligero is a minimalist Java web framework built for modern applications. Express-inspired API, virtual threads by default, and zero dependencies in the core.

Get Started View on GitHub
App.java 
import com.ligero.Ligero;
import java.util.Map;

public class App {
    public static void main(String[] args) {
        Ligero app = Ligero.create(8080);

        app.get("/hello/{name}", ctx ->
            ctx.json(Map.of("message",
                "Hello, " + ctx.pathParam("name") + "!")));

        app.start(); // 🚀
    }
}
Why Ligero

Everything you need,
nothing you don't

A modern framework designed around simplicity, performance, and zero unnecessary dependencies.

⚡

Blazing Fast

Zero dependencies in the core, served on Java 21 virtual threads. Trie-based router resolves in ~140ns with 300+ routes.

🧑‍💻

Expressive API

Fluent, Express-inspired handler API. Define routes, middleware, and error handlers with clean, readable code.

🛡️

Secure by Default

OWASP-aligned security headers, request hygiene, and CSRF protection enabled automatically. Safe out of the box.

🔄

Real-time Ready

Server-Sent Events in core, WebSockets via the Jetty engine. Build reactive features without the complexity.

🧰

Pluggable SPIs

Server engines, JSON mappers, template engines, tracing — all via ServiceLoader. Swap implementations without code changes.

📊

Observable

Per-route metrics (Micrometer), OpenTelemetry tracing, structured access logs, and a devtools dashboard.

🐳

Compile-time DI

Optional annotation processor generates explicit wiring at compile time — no reflection, no classpath scanning.

🌐

HTTP/2 & WebSocket

h2c on the Jetty engine, one line to switch. WebSocket support with the same simple handler API.
