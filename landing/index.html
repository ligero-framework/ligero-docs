App.java 
import com.ligero.Ligero;

public class App {
    public static void main(String[] args) {
        Ligero app = Ligero.create(8080);

        app.get("/hello/{name}", ctx ->
            ctx.json(Map.of("message",
                "Hello, " + ctx.pathParam("name") + "!")));

        app.start();
    }
}

Built for developers who care about quality

Every feature is optional. Every abstraction is leak-proof. Every default is secure.

⚡

Blazing Performance

Zero dependencies in the core. Virtual threads for every request. Trie-based router resolves in ~140 nanoseconds — even with 300+ routes.

📝

Express-inspired API

Clean, fluent handler API that feels familiar. Define routes, compose middleware, and handle errors with minimal ceremony.

🛡️

Secure by Default

OWASP-aligned security headers, request path hygiene, stateless CSRF — all enabled out of the box. Production-ready from the first route.

🔄

Real-time & Reactive

Server-Sent Events in core. WebSockets via the Jetty engine. Build live features without the complexity of traditional reactive stacks.

🧰

Pluggable by Design

Server engines, JSON mappers, template engines, tracing — all wired via ServiceLoader. Swap implementations without touching your code.

📊

Observable from Day One

Per-route metrics (Micrometer), OpenTelemetry tracing, structured access logs, and a live devtools dashboard. Know what your app is doing.
